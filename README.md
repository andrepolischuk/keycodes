# keycodes [![Build Status][travis-image]][travis-url]

> Key codes and names

## Install

```sh
npm install --save keycodes
```

## Usage

```js
import keycodes from 'keycodes';

keycodes('enter'); // 13
keycodes(13); // 'enter'
```

## API

### keycodes(key)

#### key

Type: `string`, `number`

Key name string or key code number.
